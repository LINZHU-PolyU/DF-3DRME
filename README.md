# DF-3DRME

The codes are for the paper "DF-3DRME: A Data-Friendly Learning Framework for 3D Radio Map Estimation based on Super-Resolution Technique," which has been submitted for possible publication.

## Code Availability

The paper is currently **under review**. The full codebase will be publicly available upon acceptance of the paper.

**Status:** Paper submitted

We are committed to full reproducibility and will release:
- Complete source code
- Pre-trained models
- Training and evaluation scripts
- Dataset processing pipelines
